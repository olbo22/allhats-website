# AllHats Brand Guidelines

## Brand Colors

### Primary

| Name | Hex | RGB | Usage |
|---|---|---|---|
| Emerald | `#10b981` | `rgb(16, 185, 129)` | Buttons, Logo, CTAs, Links, Focus States |
| Emerald Dark | `#059669` | `rgb(5, 150, 105)` | Text on light backgrounds (accessibility) |

### Secondary

| Name | Hex | RGB | Usage |
|---|---|---|---|
| Rose | `#e11d48` | `rgb(225, 29, 72)` | Accent highlights, section labels, feature icons |

### Tertiary

| Name | Hex | RGB | Usage |
|---|---|---|---|
| Cyan | `#06b6d4` | `rgb(6, 182, 212)` | Hover states, section labels, feature icons |

### Opacity Variants

Each brand color has 3 levels:

| Level | Opacity | Usage |
|---|---|---|
| Solid | 100% | Buttons, Icons, Text |
| Highlight (`-h`) | 15% | Hover backgrounds, Avatar BGs |
| Subtle (`-s`) | 6% | Card backgrounds, Pill fills |

## Typography

| Property | Value |
|---|---|
| Font | **Sora** (Fallback: -apple-system, system-ui, sans-serif) |
| Weights | 300 (Light), 400 (Regular), 500 (Medium), 600 (Semi), 700 (Bold), 800 (Extra) |
| Headlines | Bold/Extra Bold, tight letter-spacing (`-0.04em` to `-0.05em`) |
| Body | Regular 400, line-height 1.6 |
